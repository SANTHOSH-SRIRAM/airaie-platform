# Plan 04-02: Log Viewer, Node Detail Panel, and Run Actions

<objective>
Build the log viewer bottom panel with tabs (Logs, Artifacts, Cost, Timeline), auto-scroll, and copy/download actions. Build the RunNodeDetailPanel for the right panel showing node inputs, outputs, metrics, and execution history. Build the RunActionBar for the bottom bar with Cancel/Retry actions and live status. Wire all three into the existing layout routing (BottomBar contentType, RightPanel contentType) and replace the placeholder slot in WorkflowRunsPage with the real LogViewer.
</objective>

<must_haves>
- LogViewer renders monospace log output with timestamps and color-coded node names
- LogViewer has Logs/Artifacts/Cost/Timeline tabs
- Auto-scroll toggle works and defaults to on
- Copy and Download buttons export log content
- RunNodeDetailPanel shows input values, output values, metrics (Duration, CPU, Memory, Cost, Attempt), execution history timeline
- RunActionBar renders Cancel Run (red), Retry, and "Running 3/5 nodes" status indicator
- BottomBar routes to RunActionBar when contentType is 'run-actions'
- RightPanel routes to RunNodeDetailPanel when contentType is 'run-node-detail'
</must_haves>
