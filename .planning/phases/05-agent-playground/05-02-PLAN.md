# Plan 05-02: Inspector Panel, Action Bar, and Integration

<objective>
Build the InspectorPanel with decision trace timeline, live metrics, and policy status. Build the PlaygroundActionBar with Stop/Clear/Approve buttons and chat input. Wire all agent playground components into the existing layout system: InspectorPanel into RightPanel, PlaygroundActionBar into BottomBar, SessionList into SidebarContentRouter, and the AgentPlaygroundPage into App.tsx routing. After this wave, the full agent playground is functional end-to-end with all panels, actions, and navigation integrated.
</objective>

<must_haves>
- InspectorPanel shows session info, decision trace, live metrics, and policy status
- DecisionTraceTimeline renders a vertical timeline with status icons and timing
- LiveMetrics shows iterations, cost, budget, duration, timeout with live updates
- PolicyStatusCard shows auto-approve threshold and toggle
- PlaygroundActionBar has Stop Agent, Clear Session, Approve All, status indicator, chat input
- SidebarContentRouter mounts SessionList in the 'sessions' slot
- RightPanel routes 'inspector' contentType to InspectorPanel
- BottomBar routes 'agent-playground' contentType to PlaygroundActionBar
- App.tsx has lazy-loaded route for /agent-playground and /agent-playground/:agentId
- ROUTES constant includes AGENT_PLAYGROUND entry
</must_haves>
